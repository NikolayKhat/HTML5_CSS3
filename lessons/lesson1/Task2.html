Физическое форматирование текста – придает тексту некоторые визуальные отличия

Теги физической разметки

<b>...</b> - придает тексту жирное начертание

<bdi>...</bdi> - указывает фрагмент текста, который должен быть изолирован от изменения направления вывода текста. (HTML5)

<bdo>...</bdo> - устанавливает направление вывода текста (устанавливает направление вывода текста)

<i>...</i> - придает тексту курсивное начертание

<s>...</s> - перечеркнутый текст

<small>...</small> - делает текст на одну единицу меньше

<big>...</big> - делает текст на одну единицу больше

<sup>...</sup> - помещает содержимоев верхний индекс

<sub>...</sub> - помещает содержимоев нижний индекс индекс

<meter>...</meter> - используется для вывода значения в некотором известном диапазоне. (HTML5)

(значение низкое)

(значение оптимальное)

(значение высокое)

<progress>...</progress> - используется для отображения прогресса завершенности. (HTML5)

(завершение 60/100)

<span>...</span> - используется для выделения врагментов текста с последующим заданием стилевого оформления
