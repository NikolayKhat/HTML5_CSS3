Логическое форматирование текста – форматирование текста, предназначенное для удобства работы с ним поисковых систем и прочих программ

Теги логической разметки

<abbr>...</abbr> - "некая аббревиатура"

<cite>...</cite> - текст является сноской или цитатой (https://developer.mozilla.org/ru/docs/Web/HTML/Element/cite)

<code>...</code> - предназначен для хранения текста, который представляет собой программный код

Например: String str = "Hello World!"; System.out.println(str);

<strong>...</strong> - задает тексту жирное начертание

<em>...</em> - задает тексту курсивное начертание

<del>...</del> - "здесь была допущена ошибка" используется для выделения текста, который был удален в новой версии документа

<ins>...</ins> - предназначен для выделения текста, который был добавлен в новую версию документа

<mark>...</mark> - помещает текст как выделенный

<q>...</q> - выделяет текст как цитату

<samp>...</samp> - выводит текст, который является результатом выполнения программы

Например: Hello World!

<var>...</var> - используется для выделения переменных компьютерных программ

Например: String str = "Hello World!";
